L Explanation

This letter makes an l sound when all of the following conditions are met:

  • it is the last letter in a syllable,
  • it is not in the last syllable of a word, and
  • the first letter of the following syllable is a ㄹ

Examples

Korean English
연락하다 to keep in touch
연 kite

T Explanation

This letter makes a t sound when both of the following conditions are met:

  • it is the last letter in a syllable and
  • it is the last syllable in a word or the following syllable begins with a consonant sound.

Examples

Korean English
받다 to get
맛 flavor
앉다 to sit
몇 a few
놓다 to put
받았어요 got
맛있어요 delicious
앉았어요 sat
몇일 a few days
놓았어 put (past)

N Explanation

This letter makes an n sound when all of the following conditions are met:

  • it is the first letter of the syllable and
  • the previous syllable of the same word ends in an ㅇ.

Examples

Korean English
동래 Dongnae (area in Busan)
노랑 라디오 yellow radio

P Explanation

This letter makes a p sound when any of the following conditions are met:

  • the following sound is a consonant sound or
  • it is the last letter in a word

Examples

Korean English
없어요 to not have
직업 job
손잡이 doorknob

SH Explanation

This letter makes an sh sound when the following vowel is an ㅣ.

Examples

Korean English
시 hour
씨 seed
새 bird

Silent Explanation

This letter is silent when it is the first letter in a syllable.

Examples

Korean English
있다 to have
우유 milk
성공 success
